<h2>Build Setup</h2>


``` bash
## app
    # install dependencies
    npm install

    # serve with hot reload at localhost:8080
    npm run serve

    # build for production - ./app/dist
    npm run build
  
## services
    # install dependencies
    npm install

    # serve with hot reload at localhost:30000
    npm run serve

    # build for production - ./services/dist
    npm run build-ts
```
